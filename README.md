bender
======

## Settings for bash_profile/bashrc

### variables

``` bash
DOTFILES_HOME="Path to directory"
``` 

### Load

``` bash
source bender/init.sh
```

### install

``` bash
bender install
```

## Folder Structure & Command to make all the changes

### directory structure
<directory>/bundle/<project>/{autoload|symlinks}/

### commands
``` bash
bender install|remove|
```
